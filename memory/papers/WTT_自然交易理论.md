## 核心理念

### 核心三要素

| 要素 | 说明 |
|------|------|
| **空间** | 斐波那契回撤决定空间，即开平仓点位 |
| **时间** | 斐波那契时间决定时间，即交易时间点 |
| **能量** | 交易量的波动决定趋势，即多空方向 |

### 心术（70%权重）

> 天下交易，三分技术，七分心态

- 冷静分析 + 正确开单 = 盈利
- 平仓获利离场才是终点
- 心态决定能否守住利润

## 实战要点

### 1. 市场之手

- 市场有无形的手，永远按着你
- 多次清理战场，迎接新参与者
- 保持敬畏，谨慎前行

### 2. 交易如狩猎

- 等待 → 观察 → 出击
- 在变盘时间点 + 接近边缘价格 = 正确方向
