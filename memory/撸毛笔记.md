# 撸毛笔记

> 记录撸毛（AirDrop）相关知识、策略和经验

## 2026-02-13 入门指南

**来源**：X @0xbaoan「手把手教你0基础如何入门撸毛」

### 核心认知

| 项目 | 要点 |
|------|------|
| **时间成本** | 每天12-15小时在电脑前 |
| **收益周期** | 做好半年没收益的准备 |
| **本质** | 用小资金撬动更大杠杆的赌博 |
| **不确定性** | 能否拿到空投取决于投研能力、交互策略、执行力 |

### 关键风险

- **老鼠仓规则**：项目方设立特殊门槛，可能让所有努力白费
- **例子**：红石空投要求 DC 特殊角色，普通人无法获得

### 实践经验

1. **多做多完善**：不断优化交互策略
2. **思考门槛**：预判项目方可能设置的老鼠仓规则
3. **做到极致**：只完成最低要求远远不够

### 经典名言

> 「做了，不一定会有。但不做一定没有！」

## 常用工具

- 指纹浏览器（批量铺号）
- VPN/代理
- 多个钱包地址

## 注意事项

- 不要 all-in单个项目
- 分散风险
- 持续跟进项目动态
- 社区信息很重要
